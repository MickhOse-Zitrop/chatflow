# Экраны

- Главное (список чатов, поле ввода)
- Настройки (темы, параметры аккаунта)
- Личный чат
- Групповой чат

# Элементы

- Поле поиска/создания чата
- Кнопки отправки сообщения
- Статус пользователя
- Блок истории сообщений

# Состояния UI

- Empty — пустой чат
- Loading — индикатор ожидания
- Success — сообщения отображены
- Error — текст ошибки

# 3 пользовательских сценария

1. Отправка сообщения в личный чат
2. Создание группового чата и отправка сообщений
3. Просмотр истории сообщений

# Типовые ошибки

- "Введите сообщение"
- "Пользователь не найден"
- "Нет сети"
- "Сервис недоступен"
- "Слишком много запросов, попробуйте позже"
