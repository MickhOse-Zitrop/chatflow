# 1. Производительность / отзывчивость UI

- Обновление сообщений должно происходить ≤ 300 мс при стабильном соединении.
- Интерфейс чата реагирует на действия пользователя без задержек > 100 мс.

---

# 2. Надёжность

- Приложение не должно падать при отправке сообщений.
- Все ошибки должны быть обработаны и отображены пользователю в читаемом виде.

---

# 3. Хранение данных

- Избранные чаты, история сообщений и настройки сохраняются локально (кеш/база данных).
- При повторном входе данные доступны пользователю.

---

# 4. Ограничения платформы

- Web-приложение должно работать в последних версиях Chrome, Edge, Firefox.
- Не требуются права администратора для запуска.

---

# 5. Безопасность / приватность

- Личные данные пользователей не сохраняются в открытых репозиториях.
- Пароли хранятся в хешированном виде.
