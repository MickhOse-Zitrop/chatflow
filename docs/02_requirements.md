# 1. Обязательные требования (MUST)

| № | Требование                                                  | Критерий приёмки                                                                                                                                        |
|---|-------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 | Регистрация и авторизация пользователя                      | Пользователь может создать аккаунт и войти в систему; тест: новый пользователь успешно регистрируется и может войти с учётными данными                  |
| 2 | Создание личного диалога                                    | Пользователь может открыть чат с другим пользователем; тест: при выборе пользователя открывается новый чат и можно отправить сообщение                  |
| 3 | Создание группового чата                                    | Пользователь может создать группу, добавить участников; тест: группа создаётся, участники видят сообщения                                               |
| 4 | Отправка и получение текстовых сообщений в реальном времени | Сообщения отображаются у отправителя и у всех участников чата без перезагрузки; тест: при отправке сообщения оно появляется мгновенно у всех участников |
| 5 | Просмотр истории сообщений                                  | История сообщений сохраняется и доступна при повторном открытии чата; тест: сообщения остаются после перезагрузки браузера                              |
| 6 | Отображение статуса пользователя (online/offline)           | Статус меняется при входе/выходе пользователя; тест: другие пользователи видят статус корректно                                                         |
| 7 | Смена настроек пользователя (например, единицы C/F)         | Пользователь может изменить настройки; тест: настройки сохраняются и применяются к последующим сессиям                                                  |

# 2. Желательные требования (NICE)

| № | Требование                                                        | Критерий приёмки                                                                                                 |
|---|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| 1 | Поиск пользователя по имени                                       | Пользователь может найти другого пользователя; тест: поиск возвращает корректные результаты                      |
| 2 | Уведомления о новых сообщениях в чате (в пределах веб-приложения) | Пользователь видит всплывающее уведомление в окне; тест: при поступлении нового сообщения появляется уведомление |
| 3 | Возможность удаления своих сообщений                              | Пользователь может удалить сообщение; тест: сообщение исчезает из истории и у других участников чата             |
| 4 | Поддержка темной/светлой темы интерфейса                          | Пользователь может переключать тему; тест: интерфейс меняется корректно                                          |
