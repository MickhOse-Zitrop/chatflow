# 1. Название проекта

ChatFlow
---

# 2. Краткое описание проекта

ChatFlow - это веб-мессенджер для обмена текстовыми сообщениями между пользователями в режиме реального времени.
Приложение поддерживает личные диалоги и групповые чаты с сохранением истории сообщений.
Система ориентирована на быстрое, удобное и стабильное общение через браузер.
---

# 3. Пользователь (кто и зачем использует)

Целевая аудитория:

- студенты
- частные пользователи
- небольшие группы
  Цель использования:
- личное общение
- общение в групповых чатах
- быстрый обмен сообщениями без установки приложения

---

# 4. Функции (MVP)

В первую версию входят:

- Регистрация и авторизация пользователя
- Создание личных диалогов (1-на-1)
- Создание групповых чатов
- Отправка и получение текстовых сообщений в режиме реального времени
- Просмотр истории сообщений
- Отображение статуса пользователя (online/offline)

---

# 5. Что не делаем (в первой версии)

- Аудио- и видеозвонки
- Push-уведомления на устройство
- End-to-end шифрование
- Мобильное приложение

---

# 6. Платформа / стек (предварительно)

- Платформа: Web (браузерное приложение)
- Frontend:
    - Next.js
    - Tailwind CSS
    - shadcn/ui
- Backend (предварительно):
    - Node.js
    - WebSocket (real-time сообщения)
    - Реляционная база данных (например PostgreSQL)
